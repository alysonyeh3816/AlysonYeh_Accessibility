檢測方式

檢測項目：以 Tab 鍵遍歷之頁籤

  1. 以鍵盤焦點（Tab 鍵）遊走到頁籤。

  2. 確認被遊走到之頁籤報讀「索引標籤已選取」。

  3. 以鍵盤 Tab 鍵遊走頁籤，確認可進入對應的頁籤面板。

  4. 走入對應面板後，面板內資訊可取得；若面板內有互動元件，則需可操作。

  5. 再以 Tab 依序遍歷各頁籤，確認各頁籤均可執行第 (2)～(4) 項之操作。

  6. 再以 Shift+Tab 反向遊走頁籤，確認各面板與頁籤均可按邏輯被遊走。

  7. 確認頁籤組件中各元件之名稱與角色與其功能相符（正確 ARIA 揭露）。

  8. 在不同瀏覽器中，鍵盤焦點（Tab）呈現瀏覽器預設焦點樣式（如雙色黑白、雙色藍白）或作者指定之明確可見樣式。

  9. 檢查焦點指示器：區域至少有 1 CSS 像素圍繞該元素，且聚焦/未聚焦狀態間具 3:1 對比；若未達 3:1，則指示器邊線厚度需達 2 CSS 像素。

  10. 確認以 Tab / Shift+Tab 可順利離開整組頁籤組件（不會被困住）。

  11. 以上均符合則全數通過。
