檢測方式

檢測項目：純資訊輪播或跑馬燈

  1. 確認輪播/跑馬燈組件是否會動，若不會動則進入(5)。

  2. 若輪播／跑馬燈組件具有自動播放或自動移動效果，則：

    • 應確認組件中提供暫停按鈕（或刪除/隱藏按鈕，下同，統稱為「暫停/中止/隱藏元件」）。
    • 該暫停/中止/隱藏元件應為整個輪播/跑馬燈組件中第一個可聚焦元件。

  3. 若第一個可聚焦元件為整個輪播/跑馬燈組件本身，則暫停/中止/隱藏元件可作為第二個可聚焦元件確認鍵盤可操作暫停、中止、隱藏元件，並且功能與之相符。

  4. 若輪播組件提供暫停機制，則以下須擇一達成：

    • 類型一、 同時提供暫停、播放按鈕，須確認暫停與播放按鈕均可操作。
    • 類型二、 僅提供暫停按鈕，則確認暫停按鈕本身具切換功能，可進行暫停及播放功能。

  5. 以滑鼠確認各輪播圖/跑馬燈是否具有功能(功能常見如連結)。

  6. 確認具有功能後，以 Tab 鍵遊走輪播圖/跑馬燈組件各元件，確認焦點跳位順序符合邏輯。

  7. 確認各元件可操作（類型一或類型二擇一即可，惟順序必須為先確認類型一無法達成，才以類型二確認）：

    • 類型一、 若鍵盤焦點(Tab 鍵)可游走輪播圖片/跑馬燈各項目：
      • 該項目被鍵盤焦點(Tab 鍵)遊走時，被選中之輪播圖片/跑馬燈項目應可視（如圖 111）。
      • 並應主動報讀出該項目內容，且鍵盤可使用該功能（如連結，則鍵盤可以 Enter 或 Space 開啟連結）。
    • 類型二、 若鍵盤焦點(Tab 鍵)無法走入輪播圖片/跑馬燈各項目，則以 Tab 鍵遊走入向前、向後鍵或指示標籤後，應達成以下 A 或 B 任一種操作：
      • A方式：
        • 若輪播圖/跑馬燈各項目以「向前」或「向後」進行切換，則當鍵盤焦點至其上時，應報讀按鈕名稱（如：「向前」、「向後」、「上一張」、「下一張」等同義標籤皆可）。
        • 當按下上述切換按鈕時，被選中之輪播圖片/跑馬燈項目應可視。
        • 且螢幕報讀器應主動報讀出對應的圖片資訊（或者主動告知按鈕已按下）。
        • 並鍵盤焦點(Tab 鍵)可遊走至對應之唯一輪播圖片/跑馬燈項目。
        • 且鍵盤操作須可使用該功能（如連結，則鍵盤可以 Enter 或 Space 開啟連結）。
      • B方式：
        • 若輪播圖／跑馬燈項目以指示標籤進行切換，則當鍵盤焦點停留於指示標籤時，螢幕報讀器應朗讀該項目的項目值（例如：「項目一」、「項目二」）或該項目的名稱。
        • 當輪播圖／跑馬燈項目被選取時，該被選取的項目應在視覺上清晰可見。
        • 項目被選取時，螢幕報讀器應主動朗讀對應的狀態提示，例如：「已按下」、「已選取」等。
        • 使用者應可透過 Tab 鍵將鍵盤焦點移動至該被選取的唯一輪播圖／跑馬燈項目。
        • 若該項目為可互動元件（例如連結），則使用者應可透過鍵盤操作（如 Enter 或 Space）啟動其功能（如圖 112）。

  8. 承前項(7)，鍵盤可操作類型一或類型二（不考量報讀）。

  9. 承項目(7)，若選擇類型一，則檢查當鍵盤焦點(Tab)停駐在圖片輪播圖片/跑馬燈項目，該圖片/跑馬燼項目會暫停，此時可清楚看見鍵盤焦點指示符。

  10. 再次按鍵盤焦點(Tab)，鍵盤焦點可移動並選取下一項。

  11. 鍵盤焦點(Tab)取到各可聚焦元素時，確認在不同瀏覽器中，鍵盤焦點皆呈現瀏覽器預設的指示樣式（例如雙色黑白框、雙色藍白框）或由網頁作者指定的明確可見變化的視覺樣式。

  12. 檢查焦點指示器的區域至少有 1 CSS 像素圍繞該元素，並檢查其在聚焦與未聚焦狀態之間是否具有 3:1 對比值（若未達 3:1，則檢查邊線厚度是否有 2 CSS 像素）。

  13. 確認輪播圖、跑馬燈各元件之名稱及角色與其功能相符：

    • 如輪播圖片角色為圖片。
    • 如暫停鍵、切換項目向前、向後則角色為按鈕。
    • 具連結功能之輪播圖/跑馬燈項目則角色為連結。
    • 類型二之 A 按下向左、向右鍵後應主動報讀出對應的圖片資訊（或者主動告知按鈕已按下）。
    • 類型二之 B 指示標籤被選擇後應主動報讀相應提示（如：已按下、已選取）。

  14. 確認以鍵盤 Tab 鍵及 Shift+Tab 鍵可遊走輪播/跑馬燈組件時，最終可順利離開整組組件。

  15. 以上均符合則全數通過。

註一：若第一個鍵盤焦點是整組輪播，並且能報讀整組件為輪播，則暫停、中止或直接在視覺且無障礙均隱藏的機制（如暫停鈕、隱藏鈕、移除鈕），可以為第二個焦點。

註一的白話文說明：有些程式會用一個大標籤包裹整個輪播或跑馬燈（或在輪播或跑馬燈組件前設一個說明標籤），然後讓那個大標籤（或說明標籤）可聚焦並報讀「輪播圖」。這樣就會導致暫停鈕、隱藏鈕、移除鈕會是輪播組件第二個可聚焦的元件。
